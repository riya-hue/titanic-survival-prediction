# Titanic Survival Prediction

## Project Overview
Predict whether a passenger survived the Titanic disaster using passenger data (age, sex, class, siblings, fare, etc.). This project includes data preprocessing, training multiple ML models, and visualizing insights through exploratory data analysis (EDA).

## Dataset
[Titanic Dataset](https://www.kaggle.com/c/titanic)  
- CSV File: Titanic-Dataset.csv  
- Features: PassengerId, Survived, Pclass, Name, Sex, Age, SibSp, Parch, Ticket, Fare, Embarked

## Objectives
- Predict survival outcome of passengers
- Compare multiple ML models: Logistic Regression, Random Forest, KNN
- Handle missing data and encode categorical features
- Explore feature distributions and correlations
- Visualize data through histograms, scatter plots, and heatmaps

## Technologies Used
Python, Pandas, Scikit-learn, Seaborn, Matplotlib
